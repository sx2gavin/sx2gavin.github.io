I created this website as my personal blog and portfolio. It uses Jekyll static website framework and it's hosted using Github Page. I wanted to create this website for so long and I was always trying to postpone it. Right now this is finally here and I'm very excited to start using this website as my blog and portfolio.

Initially I wanted to create a website that showcases my personal projects and my games, but I couldn't decide what technology to use for this website. When I saw that github allows you to host your own personal website with their domain, with just a little bit of setup, I was excited to try this out.

I wasn't satisfied with the fact that I or other people have to use `sx2gavin.github.io` to access my website, first of all it's hard to remember, and it's also not professional at all. So I applied for a domain name from namecheap.com, recommanded by one of my co-workers. I linked my website to my new domain name: `gavinluo.com`. It's so good now because it's really professional and easy to remember!
