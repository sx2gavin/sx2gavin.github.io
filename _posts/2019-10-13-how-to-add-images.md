I am trying to add some images to my post that showcases my games. I did some research online and this is a short tutorial for me to remember how to add images in Jekyll posts.

First of all, make a `assets/images` folder in the directory and put some images in there. For example, I put a screenshot of my game, MrBlocker under the created folder. When you would like to use this image in a post, add the image like this:

```
![MrBlocker Screenshot](/assets/images/mrblocker_screenshot.png)
```
